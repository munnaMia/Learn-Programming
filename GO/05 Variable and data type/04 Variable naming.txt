Variable nameing rules in Golang:
=================================

    01. Variable name always start with a letter(name, age, status) or an underscore character (_)(_active, _name).
    02. Variable name cannot start with a digit(0name, 90age)
    03. Variable name can only contain alpha-numeric characters and underscores (a-z, A-Z, 0-9, and _ )
    04. Variable names are case-sensitive (name, Name and NAME are different variables)
    05. There is no limit on the size of the variable name in Golang
    06. Variable name cannot contain spaces(name is, age is)
    07. The variable name cannot be any Go keywords(for, if, else, go, chan)


Camel Case naming:
==================
    myNameIS , hisAgeIS

Snake Case naming:
==================
    my_id_is, active_status, user_id

Pascal Case naming:
==================
    UserId, ActiveStatus
